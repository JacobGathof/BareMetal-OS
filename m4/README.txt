Build and run project by running compileOS.sh
Project uses qemu, bcc, as86, and ls86

Running the project will launch the shell.
Enter a string and press enter.
There are several commands available:
	"type <arg>" - print out the contents of the file arg
	"execute <arg>" - execute the program arg
	"create <arg>" - the user will be prompted for input for a new file
	"copy <arg1> <arg2>" - copy the contents of arg1 into a new file arg2
	"delete <arg>" - delete file <arg>
	"dir" - list the directory and their sector sizes
